SHABR Project

For this project I had to make a website for a fake Sky Hot Air Balloon Ride business

The purpose of this project was to learn Bootstrap. It was very challenging for me. I watched a lot of videos, and went on a lot of websites to understand it. It was a nice experience for me I did learn a lot like how to make a carousel, jumbotron, navbar, etc, these are all components of Bootstrap.

I have seen many of these components on websites I regularly go on, but I never knew how it was coded until I tried it out myself.
